Risk Measures/Metrics:
PSP1%: Valor de ganho/(perda) quando o spread aumenta em 1%.
o	PSP1% > 0 (Short risk) -> Ganho
o	PSP1% < 0 (Long risk) -> Perda
•	PSP1% está associado ao spread da CDS curve de 5Y; Se o spread da curva é mais elevado, então o PSP1% também é mais elevado (proporcionais).
•	CR Delta está associado ao spread das CDS curves para todos os tenors.

ISR (Issuer Risk/Default Risk): Valor de perda/(ganho) para 0% de recovery no caso de default.
o	ISR > 0 (Long risk) -> Perda
o	ISR < 0 (Short risk) -> Ganho
•	ISR (CDS) = ISR (Bond) -> Interpretação equivalente em termos de sinal

Curvas (CDS): 
•	Seniority: Hierarquia de pagamento em caso de default:
o	PRE – Perpétua;
o	SEN – Seniority/Senior;
o	SUB – Subordinada;
o	…
o	Tier1

EXCEL Shortcuts:
Dominar atalhos no Microsoft Excel muda completamente a velocidade de trabalho.
Seguem os mais úteis mesmo no dia-a-dia (especialmente com Excel em português).

🔎 Navegação e procura
•	Ctrl + L → Localizar
•	Ctrl + U → Substituir
•	Ctrl + G → Ir para (Go To)

✏️ Edição básica
•	Ctrl + C / V / X → Copiar / Colar / Cortar
•	Ctrl + Z / Y → Desfazer / Refazer
•	Ctrl + Enter → aplicar o mesmo valor em várias células selecionadas
•	F2 → editar célula rapidamente

⚡ Seleção rápida (MUITO importante)
•	Ctrl + Shift + ↓ / ↑ / → / ← → selecionar até ao fim dos dados
•	Ctrl + Espaço → selecionar coluna inteira
•	Shift + Espaço → selecionar linha inteira
•	Ctrl + A → selecionar tudo

📊 Trabalhar com dados
•	Ctrl + T → transformar em tabela
•	Alt + ↓ → abrir dropdown de filtro
•	Ctrl + Shift + L → ativar/desativar filtros

🧮 Fórmulas
•	Alt + = → soma automática
•	F4 → fixar referências ($A$1)
•	Ctrl + ` → mostrar/ocultar fórmulas

📄 Formatação
•	Ctrl + 1 → abrir formatação de células
•	Ctrl + B / I / U → negrito / itálico / sublinhado
•	Ctrl + Shift + $ → formato moeda
•	Ctrl + Shift + % → percentagem

🧭 Navegação rápida dentro da sheet
•	Ctrl + Home → ir para A1
•	Ctrl + End → ir para última célula usada
•	Ctrl + Page Up / Page Down → mudar de folha

🧱 Linhas e colunas
•	Ctrl + + → inserir linha/coluna
•	Ctrl + - → apagar linha/coluna

🧠 Truques avançados (muito úteis)
•	Ctrl + D → copiar de cima para baixo
•	Ctrl + R → copiar da esquerda para a direita
•	Alt + Enter → quebra de linha dentro da célula

💡 Os 5 que mais fazem diferença (prioriza estes)
•	Ctrl + L (procurar)
•	F2 (editar rápido)
•	Ctrl + Shift + setas (navegação brutal)
•	Ctrl + T (tabelas)
•	F4 (fixar células)
